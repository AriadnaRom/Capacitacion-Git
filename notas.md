# Notas de Git

1. Primero se usa git init para inicializar el repositorio de forma local.

2. Luego se crea el archivo README.md y se escribe la información del proyecto.

3. Después se usa git add . para añadir los archivos de forma local.

4. Se realiza el commit con git commit -m "Primer commit".

5. Se crea el repositorio remoto en GitHub, se escoge la rama principal y se usa  
   git push para subir los commits al repositorio remoto.
